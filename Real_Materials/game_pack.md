# PMO Game Pack

## Game 1: PMO Decision Quest

Students choose the best PMO action in five scenarios. Each correct answer unlocks a badge.

### Question 1

The sponsor asks the team to start coding immediately. What should the PM do first?

- Start coding
- Define scope and stakeholders
- Buy tools
- Ask for more budget

Correct answer: Define scope and stakeholders  
Badge: Scope Master

### Question 2

Your team submits a WBS that only says: code, test, deploy. What is the problem?

- It is too detailed
- It is not deliverable-oriented
- It has too many stakeholders
- It is already complete

Correct answer: It is not deliverable-oriented  
Badge: WBS Architect

### Question 3

The first schedule takes 15 weeks, but the deadline is 12 weeks. What is the best action?

- Remove testing
- Hide the delay
- Parallelize work and prioritize scope
- Add optional features

Correct answer: Parallelize work and prioritize scope  
Badge: Schedule Optimizer

### Question 4

Week 6 dashboard: planned progress 50%, actual progress 38%, actual cost above plan. What is the diagnosis?

- Project is healthy
- Late and over budget
- Ahead of schedule
- Ready to close

Correct answer: Late and over budget  
Badge: Dashboard Detective

### Question 5

Week 8 crisis: database integration fails and the team is overloaded. What should the PMO do?

- Add more features
- Stop testing
- Freeze scope and focus critical deliverables
- Hide the problem

Correct answer: Freeze scope and focus critical deliverables  
Badge: Crisis Controller

## Game 2: Dashboard Detective

Students inspect the week 6 dashboard and diagnose the project.

### Dashboard Data

- Planned progress: 50%
- Actual progress: 38%
- Planned cost: 20,000 USD
- Actual cost: 24,000 USD
- Critical bugs: 12
- Completed modules: 2 of 4 planned modules

### Questions

1. Actual progress is 38% while planned progress is 50%. What does this show?
   - Correct answer: Behind schedule

2. Actual cost is 24,000 USD while planned cost is 20,000 USD. What does this show?
   - Correct answer: Over budget

3. There are 12 critical bugs at week 6. What risk is visible?
   - Correct answer: Quality risk

4. Which action is strongest for the PMO?
   - Correct answer: Freeze critical scope and fix integration
