# PMO AI Teaching Module

React/Vite version of the AI-assisted gamified teaching module:

**From Plan to Control: Managing a Project Like a Real PMO**

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub

Push this folder to a GitHub repository. For GitHub Pages, connect the repository to a static hosting workflow or deploy the generated `dist` folder after running `npm run build`.

## Included Material

- React module dashboard
- Five PMO quests
- Instructor runbook
- Student worksheet content
- Quiz and rubric
- Explanatory video structure
- Dashboard CSV in `public/dashboard_template.csv`
