# Quiz and Exit Ticket

## Formative Quiz

### Question 1

What should a project manager do first when receiving a new project request?

A. Start implementation  
B. Define scope and stakeholders  
C. Buy project software  
D. Assign random tasks  

Correct answer: B

### Question 2

Which statement best describes a WBS?

A. A list of employee responsibilities  
B. A deliverable-oriented decomposition of project work  
C. A budget approval form  
D. A meeting agenda  

Correct answer: B

### Question 3

The first project schedule takes 15 weeks, but the deadline is 12 weeks. Which action is most professional?

A. Remove all testing  
B. Hide the delay  
C. Re-plan using parallel work, scope prioritization, and resource adjustment  
D. Add more features  

Correct answer: C

### Question 4

If actual progress is 38% while planned progress is 50%, what does this suggest?

A. The project is ahead of schedule  
B. The project is late  
C. The project is complete  
D. The budget is automatically healthy  

Correct answer: B

### Question 5

If actual cost is higher than planned cost, what should the PM investigate?

A. Cost overrun causes  
B. The color of the dashboard  
C. Only the project logo  
D. How to avoid reporting the issue  

Correct answer: A

### Question 6

What does SPI below 1 usually indicate?

A. The project is ahead of schedule  
B. The project is behind schedule  
C. The project has no cost problem  
D. The project is closed  

Correct answer: B

### Question 7

What does CPI below 1 usually indicate?

A. Cost performance is inefficient  
B. The project is under budget  
C. The schedule is complete  
D. The scope has no risk  

Correct answer: A

### Question 8

During a crisis, which action is most responsible?

A. Freeze or prioritize scope and focus on critical deliverables  
B. Add non-essential features  
C. Stop communicating with the sponsor  
D. Ignore bugs  

Correct answer: A

### Question 9

Which activity is an example of project control?

A. Reading the project title  
B. Comparing actual performance to plan and taking corrective action  
C. Choosing the font for slides  
D. Naming the project team  

Correct answer: B

### Question 10

What should a PMO recovery briefing include?

A. Problem, impact, corrective action, and sponsor decision needed  
B. Only jokes and excuses  
C. Only screenshots  
D. Only team complaints  

Correct answer: A

## Exit Ticket

Students answer individually:

1. What is one difference between monitoring and controlling?
2. What was the most important corrective action in your group plan?
3. How did AI help your team during the activity?
4. What should a project manager never hide from the sponsor?
