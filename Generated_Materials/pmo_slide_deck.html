Managing a project like a real PMO

Case: Smart Campus Mobile App

Today You Are the PMO

The university president wants a Smart Campus Mobile App launched before the next semester.

  • 12 weeks
  • 40,000 USD budget
  • Limited technical team
  • Demo expected by week 6

Learning Objectives

  • Build a deliverable-oriented WBS.
  • Convert WBS items into a schedule.
  • Optimize execution under constraints.
  • Analyze dashboard indicators.
  • Propose corrective action during a crisis.
  • Present a professional PMO briefing.

The Project Case

App Features

  • Student login
  • Course schedule
  • Exam calendar
  • Notifications
  • Room reservation
  • Complaint/request system
  • Admin dashboard

Constraints

  • 12-week deadline
  • 40,000 USD budget
  • Database integration risk
  • First demo by week 6

Level 1: Project Launch

First decision: What should the PM do first?

  • A. Start coding immediately
  • B. Define scope and stakeholders
  • C. Buy software tools
  • D. Ask for more budget

Correct direction: define scope and stakeholders.

Level 2: WBS Builder

A WBS is a deliverable-oriented decomposition of project work.

Major DeliverableExample Sub-deliverables
RequirementsStudent interviews, admin interviews, functional requirements
DesignWireframes, UI prototype, database design, API design
DevelopmentAuthentication, schedule module, notification module, admin dashboard
TestingUnit testing, integration testing, user acceptance testing

Level 3: Schedule Rescue

The first plan takes 15 weeks. The sponsor requires 12 weeks.

Possible Recovery Actions

  • Parallelize UI and backend work.
  • Reduce non-critical scope.
  • Add temporary development support.
  • Delay advanced features.
  • Increase testing automation.

Level 4: Dashboard Detective

Diagnosis: late, over budget, and technically unstable.

Level 5: Crisis Controller

At week 8:

  • Database integration fails.
  • QA reports many bugs.
  • The sponsor refuses to extend the deadline.
  • The team is overloaded.

Prepare a three-minute recovery briefing.

PMO Recovery Briefing

  • What happened?
  • What is the impact?
  • What corrective action is proposed?
  • What decision is needed from the sponsor?
