Module Metadata

Title: From Plan to Control: Managing a Project Like a Real PMO

Topic: Project Management, WBS, scheduling, monitoring, control, risk, corrective action

Level: Licence / Master

Duration: 180 minutes

License: CC-BY-NC recommended

Learning Objectives

  1. Explain the project management cycle from initiation to closing.
  2. Build a deliverable-oriented WBS.
  3. Convert a WBS into tasks, dependencies, milestones, and responsibilities.
  4. Optimize a project schedule under constraints.
  5. Interpret dashboard indicators and diagnose project health.
  6. Propose corrective actions and present a PMO recovery briefing.

Gamification
