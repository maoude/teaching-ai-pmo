# Real Teaching Materials

These files are production-ready assets for the PMO AI teaching module.

## Files

- `lecture_delivery_script.md`: instructor script for the full 3-hour lecture.
- `explanatory_video_script_full.md`: 8-10 minute final project video script.
- `micro_video_scripts.md`: scripts for three short explainer videos.
- `genially_storyboard.md`: screen-by-screen storyboard for the interactive simulation.
- `quizizz_import.csv`: quiz bank formatted for import or manual entry.
- `question_bank.json`: structured question bank for future React quiz interactivity.
- `activity_cards.md`: printable group activity cards.
